# IPL Match Ticket Booking System

A web application developed using Flask and SQLite for booking IPL match tickets.

## Features

- User Registration
- User Login
- View IPL Matches
- Seat Category Selection
- Seat Selection
- Book Tickets
- View My Bookings
- View All Bookings

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap
- JavaScript
